## 매매포지션

### 1️⃣ 스캘핑
- 호가창 매물돌파 : 큰거래대금과 함께 돌파하는 자리
- 신고가, 전고돌파

<br/>

[[TOP]](#index)

---
### 2️⃣ 종가베팅
- 새로운 재료 : 지속성 여부
- 세력이 매수 : 외인/기관의 매수 흔적

<br/>

[[TOP]](#index)

---
### 3️⃣ 시초베팅
- 최근 신고가종목이 기간/물량 소화이후 재돌파하는 자리
- 세력의 매집한 거래량이 빠져나가지 않은 종목

<br/>

[[TOP]](#index)

---
### 4️⃣ 단기스윙
- 최근 관심받은 테마가 급등후 눌림자리에서 횡보
- 개인의 물타기와 손절이 거의 마무리 되고 거래가 거의 없다가 다시 매수세가 들어오는 자리
- 가격조정/기간조정이 충분히 이루어진 자리
