### (스윙) [일]3번자리_바닥포착
> _급락후에 장기박스권을 돌파하고 쉬어가는 밥그릇3번자리를 포착_ 
<br/>

- **조건식 생성** : (plus and) 🅿️ / 🅾️ (or group)

| 지표 | 내용 |
|:---:|-----|
| A | 신고가:[일]0봉전 고가가 20봉이내 240봉 신고가 발생 |
| B | 기간내 주가위치:[일]0봉전 10봉 이내에서 종가가 (최고가-최저가) 변동폭이 최저가 대비 10% 이상 |
| C | 52주 최고가대비 -20%이하 |
| D | 신고가:[일]0봉전 종가가 3봉중 신고가 |
| E | 주가등락률:[일]1봉전(중) 종가대비 0봉전 종가등락률 5%이상 |
| F | 거래량비율(n봉):[일]1봉전 거래량 대비 0봉전 거래량 비율 200%이상 |
| G | 신고가:[일]1봉전 종가가 3봉중 신고가 |
| H | 주가등락률:[일]2봉전(중) 종가대비 1봉전 종가등락률 5%이상 |
| I | 거래량비율(n봉):[일]2봉전 거래량 대비 1봉전 거래량 비율 200%이상 |

- **조건식 실행** <br/>
__A and B and C and ((D and E and F) or (G and H and I))__
  - **대전제** : 거래량 증가, 변동률 20%이상, 정배열
  - **이탈여부** : 일봉상 5이평과 장기이평간의 이격도가 5%이내
  
- **조건검색기 성과검증** :
  [📈 검색종목 차트보기](./docu/종베_3번자리_종가베팅/)
