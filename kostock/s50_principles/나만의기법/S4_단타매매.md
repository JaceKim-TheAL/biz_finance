## 4️⃣단타매매

### 분봉추세단타
  - **이평배열** : **[차트]** `단기역배열`(5이평<10이평<20이평) / `단기정배열`(5이평>10이평>20이평)_
  - **추세반등** : **[매수]** 역배열에서 하락추세선 or 10이평을 상승돌파후 지지, 이평선이 정배열로 전환
  - **상승추세** : **[매수]** 정배열에서 5일선을 타고 상승하면 매도 금지
  - **하락반등** : **[매도]** 종가상 5이평 하락돌파시(1차매도), 10이평 이탈시(2차매도), 20이평 이탈시(전량매도)
  - **낙폭잡기** : **[매수]** 역배열상태 20이평 아래(이격도커짐)에서 양봉(C>이전봉중심)이 발생하면 반등시도
  - **손절라인** : **[매도]** 반등시도한 시가를 이탈시 or 20이평을 하락이탈하면 손절(전량매도) <br/> 
  - **시간별/차트별** 
    - 장초에는 1분봉 차트, 장중에는 3분봉/5분봉 차트
      - 종목마다 반응하는게 다르지만, 이격도에 민감한 분봉차트를 주로 참조!!
    - 장초에는 전고돌파, 10시 이후에는 눌림공략(쌍바닥)
  - **매매전략**

   | 전략기준 | 상승구간  | 하락구간 |
   |---------|--------|---------|
   | **이평배열** | ● 정배열🟨(5이평>20이평)     | ● 역배열🟫(5이평<20이평) |
   | **추세선긋기**   | **[상승추세선]** : 저점간 연결선을 고점에 이동 **(예측가능)** <br/> ● 저점이 상승한만큼 고점도 상승 | **[하락추세선]** : 고점간 연결선, 끝은 알수없다 **(예측불가)** <br/> ● 고점이 하락하며, 추세선을 돌파후 안착까지 무조건 기다려라 |
   | **추세전환** | ● 주가는 하락추세선 돌파후 지지 <br/> ● 이평선들이 밀집하고, 20이평을 돌파후 안착 확인 | ● 고점에서 이격이 최대벌어진후 하락, 거래량도 최고찍고 함께 감소 <br/> ● 10이평과 20이평을 단계적으로 하락돌파 |
   | **주가방향** | ● 📈상승시 주가는 상승하는 5이평을 타면서 함께 올라간다 <br/> ● 매물대 및 전고까지의 상승공간 확인하고 목표가 설정 | ● 📉하락시 주가는 5이평이 하락하며 <br/> ● 파동은 10/20이평 터치만 할 뿐 절대 안착하지 못한다 |
   | **매수타점1** | **[돌파매수]** <br/> ● 전고점을 강한거래량과 함께 돌파 <br/> ● 매물대를 소화하기 위한 강한거래량 동반확인 | **[돌파매수]** <br/> ● 하락추세선을 강한거래량과 함께 돌파 <br/> ● 큰매수물량이 없다면 지지선을 믿고 과감하게 매수 (이탈시 칼손절) |
   | **매수타점2** | **[눌림매수]** <br/> ● 상승1파후 쉬어가는 첫눌림 자리 (5이평/10이평 터치) <br/> ● 상승3파후 횡보하면 (20이평/볼벤하한선 기다림) | **[눌림매수]** <br/> ● 전저점이탈시 대칭기법(직전고점-이탈저점)으로 반등지점 예상 <br/> ● 5이평 이격도 침체변곡지점에서 거래량과 함께 양봉 출현 (낙폭단타) |
   | **매도타점1** | ● 5이평 이격도 과열변곡지점 (매도후 반등재매수) <br/> ⇒ 이격벌어지면 프로그램 매도쏟아짐  | ● 첫돌파이후는 전고점부근 일부매도 <br/> ⇒ V자반등 기대하지말고, 매물대가 소화되어야 본격 상승하기때문 |
   | **매도타점2** | ● 10이평을 이탈하면 절반매도 <br/> ● 20이평을 이탈하면 전량매도 | ● 하락구간에서는 이평터치가 잘먹힘 <br/> ● 즉, 와리가리하며 한계단씩 이평돌파하며 정배열전환 |

- **[5이평] 매수/매도 체결강도의 힘을, [20이평]은 추세의방향** (각도가 중요)
- 상승/하락시 이평간 이격도가 확장되고, 추세를 바꾸기전에 이평간 이격도가 수렴
  - 상승구간에서 이격도 최대는 매도타점, 하락구간에서 이격도 최대는 매수타점
  - 원샷원킬보다는 추세를 믿고, 이평터치로 매수매도를 하며 끌고가는 매매가 수익을 극대화 
- 추세를 확인하고 길게보면 크게 먹는다.
  - 첫번째 추세반등지점은 신뢰도가 낮고, 얼마든지 역전/반등이 가능하므로 두방향을 모두 고려
  - 분봉차트에서 **5이평 이격도** 참조 : **과열변곡(매도), 침체변곡(매수)**

### 지지와 저항

| No  | 지지 | 저항 |
|:---:|-----|-----|
| ❶ | 가격이 하락하다가 더이상 내려가지않고 버텨내는 가격 구간 | 상상하던 가격이 더이상 위로 진행하지 못하고 멈추거나 되돌려지는 구간 | 
| ❷ | 하락하는 추세속에 매수세가 매도세를 압도하여 가격하락을 막아내는 역할을 하는 자리  | 본질적으로 공급이 수요를 압도하는 자리 |    
| ❸ | 가격을 더 내려가지 못하게 돈을 쓴 구간 | 물렸던 개미들이 본전가에서 투매를 하는 구간 |
| ❹ | 전저점 / 기준봉의 저가 or 시가 | 전고점 / 기준봉의 고가 or 종가 |
| ❺ | 단타시 매수자리 | 단타시 매도자리 |

### 지지대와 저항대

- 단순한 "선"이 아닌 "대"로 이해해야 한다.
  - 시장 참여자들의 주문은 한가격에 몰려 있는게 아니라 범위안에 흩어져 대기하기 때문
  - 우측 연장으로 봤을때 뒤의 주가에 지속적인 영향을 미친다.
- 첫번째 지지, 첫번째 저항이 가장 강력하고
  - 3번 이상 두드리면 흐름이 바뀔 가능성이 크다
- 지지와 저항은 현재 주가를 기준으로 가장 가까운 캔들부터 멀리 있는 캔들의 순서로 분석한다.
  - 차트를 볼때 왼쪽에서 오른쪽으로 보면서
  - 급반등이 나오는 지점이나, 여러번 반등이 나오는 지점을 확인한다
- 단타를 할때 유리한 지점은...
  - 저항구간보다 지지구간에서 매수를 하는게 유리하다
  - 왜냐하면, 주식은 싸게 사서 비싸게 팔아야 하기때문
  - 주가가 빠졌다가 점점 올라오면, 고가에서 물린 개미들의 본전심리로 투매가 나올 확률이 높다
- 매집봉
  - 상승할때 거래량이 많아졌는데, 
  - 하락할때 거래량이 크지않다면..
  - 주가를 들어올린 세력이 빠져나가지 않았다는 의미

### 상승신호탄(상한가분석)
  - 1분봉 : 거래량200% + 상승률1%
  - 3분봉 : 거래량200% + 상승률3%
  - 5분봉 : 거래량200% + 상승률5%
  - **대상승인 경우 거래량이 500%이상**인 경우가 많다.

### 돌파눌림
  - 상승초입 or 돌파타점을 잡지 못하면 **추격매매(뇌동)하지 않는다.**
    - 즉, 계속되는 빨간봉에서 매수금지 (반드시 기준봉을 확인할것)
    - 상승시작과 상승의 끝을 확인후(기준봉), 4등분선 2Q 눌림기다린다.
  - 진짜 돌파는 반드시 돌파선을 지지 확인시켜준다.
    - 반드시 큰거래대금이 동반되어야 한다. 
    - 반드시 매수세의 주체를 확인하자.
  - **상승시작가를 이탈하면 무조건 손절**

### 익절원칙
  - 연속상승시 **거래량이 줄거나, 긴꼬리발생, 음봉**이 만들어지면 무조건 반절 매도(50% 익절)
  - **10일선**이 무너지면 반절 매도(25% 익절)
  - **20일선**이 무너지면 전량 매도(25% 익절)후 매매금지!

### 손절원칙
  - 매수전 손절라인을 긋는다
  - 진행중에 **손절라인을 절대 내리지 않는다!**
  - 매물대를 **돌파하면 손절라인을 올린다.** 
  - 매수직후 **예상과 다르게 움직이면 전량매도** 후 타점을 다시잡는다.
