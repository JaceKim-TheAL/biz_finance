## 신호등라인
> 상승종목의 당일 시가 기준으로 저항/지지 라인

[수식]
```js
신2 = dayopen()*(100+상승률1)/100;
호2 = dayopen()*(100+상승률2)/100;
등2 = dayopen()*(100+상승률3)/100;
라 = dayopen()*(100+상승률4)/100;
인 = dayopen()*(100+상승률5)/100;

DH = dayhigh();
ML = if(DH>신2, 신2, if(DH>호2, 호2, if(DH>등2, 등2, 라)));
신 = if(DH>신2, 신2, ML);
호 = if(DH>호2, 호2, ML);
등 = if(DH>등2, 등2, ML);
```

[지표조건설정]
```js
상승률 = [0, 2, 5, 10, 21]
```
