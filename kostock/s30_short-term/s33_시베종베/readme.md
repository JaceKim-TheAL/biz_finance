## 종가베팅 Tip 

### 👉 종목선정
- **거래량** : 1년 기간중 최고 거래량 **(240최고거**)
- **회전율** : 유동거래량 대비 **회전율상위 종목**
- **보조지표** : **볼린저밴드 상단 돌파** 윗꼬리, 단 꼬리가 몸통보다 짧아야 함.
  - 최고종목 : 볼벤의 상한선을 위로 찌르고, 찌르고를 반복하며 상승 
  - 최악종목 : 볼벤의 하한선을 아래로 이탈, 이탈하기를 반복하며 하락

### 👉 종베원칙
- 자동 손절가 준수 : **-3%면 무조건 손절**
- 종가매매종목 **최소 2~3개**
- 미련 갖지 말기!
- 만물상 만들지 않기! 

### 👉 종베종목 제외
- [x] 아무리 강한테마 종목이라도, **거래대금과 재료** 가 받쳐주지 않으면 진입 보류
- [x] 아무리 수급이 좋아도, **공매도가 크게 들어온 종목은 보류**
- [x] 음봉하락에 쌍끌이매도가 나오더라도, 또 오후장에 상승 혹은 횡보하여도, **5일선이 무너지면 보류**
  - 단 전날 쌍끌이매수가 크고 5일선이 무너지지 않는다면 터치후 반등 예상
- [x] 아무리 뉴스가 좋고 재료가 좋고 분봉차트가 좋아도, **일봉차트가 최근들어 너무 고점종목은 보류 (즉, 신고가 여러번)**
  - 차라리 시초베팅으로 대응하는게 낫다. ⇒ 눌림후 반등시점에 매수세가 강하게 들어오면 진입
  - 가더라도 장초에 눌러주고 가더라. ⇒ 즉, 시초에는 갭하락으로 개미들 물량뺏기. 
  - 안털린재간이 없다! 피도눈물도 없는 주식시장!
