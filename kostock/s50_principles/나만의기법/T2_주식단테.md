## ▶️주식단테의 기법특강


### 매매기법
- [눌림목강의](https://www.youtube.com/watch?v=SVzarQfimWQ)
- [역매공파기법](https://www.youtube.com/watch?v=fqaOvEhGrvQ)
  - 1.역배열 : 가격이 싼 저평가된 주식을 찾기 위함
  - 2.매집봉 : 수급의 주체가 될만한 거래량이나 파동이 들어온것을 확인
  - 3.공구리 : 밑에 지지라인이 형성되었는지 확인
  - 4.파란점선 : 충분한 힘이 응축되어 폭발할 단계인지 확인 (볼린저밴드의 수치 조정)
  - 5.112일선 : 60일선(추세선)까지 정배열로 돌렸는지 확인
    - 장기이평은 역배열, 단기이평은 정배열
    - 즉 60일선이 지지가 되어있는 상태
