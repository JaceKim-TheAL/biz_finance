### 마인드셋

- 신의영역에 도전하지 않는다. 
  - 저점에서 사서, 고점에서 파는 것은 불가능의 영역이다.
  - 왜냐하면 지나봐야 알수있기 때문
  - 불확실하고 쓸데없는데 에너지를 뺏기지 말자!
  - 하루에 10종목 매매보다, 확실한 1~3종목에 집중하자!

- 시장의 기준과 생각이 중요하다.
  - 시장이 반응하지 않으면 나의 생각은 보류한다.
  - 내가 바라보는 관점보다, 시장이 바라보는 관점에서 고민한다.
    - 주가를 핸들링하는 세력이 어떻게 생각하는지를 생각한다.
    - 또한 개미들이 생각하는 것과 반대로 생각해보자.
    - 세력은 개미들을 누구보다 잘안다.

- 최선이 아니라면, 차선을 선택하고 만족하자
  - 세력보다 조금 비싸게 사고, 
  - 세력보다 조금 더싸게 판다.
  - 즉, 세력을 따라 한다. 사면 따라사고, 팔면 따라판다.

- 티끌모아태산 전략
  - 한방을 노리지 말고, 시간을 아끼자!
  - 도파민과 같은 큰수익에 현혹되지말고, 시간대비 효율적인 매매를 하자!
  - 100억,200억 고수를 부러워하지 않는다. 50억만 벌어도 죽을때까지 다 못쓰고 죽는다.
  - 복리의 마법과 한계
    - 2^10 == 1024       : 복리의 마법은 엄청나다 
    - 2^20 != 1024*1024  : 복리도 한계가 있다. 왜냐하면 시장과 나의그릇에 한계가 있기때문이다.
    - 즉, 시간에 쫓기지말고, 조급해하지 말자!

- 대마불사가 아니라 대마죽으면 즉사
  - 과신하지말고 몰빵하지마라
  - 몰빵했다면 생각대로 안움직이면 무조건 정리, 신속이 생명!
  - 물리는 손실보다 물려있을 시간이 더 아깝다는 것을 잊지마라.

- 심플 is 베스트!
  - 복잡한 기법보다, 단순하고 직관적인 기법으로 빠르게 의사결정하자
  - 종목을 쉽게 찾고, 수익을 내는 패턴을 만들자!
  - 가능하면 대장주만 매매한다. 즉, 돈이 몰리고 사람의 관심이 집중되는 종목만 매매한다.
  - 당일 지수흐름과 종목의 매수주체 즉 거래원이 누구인지 파악하고 매매한다.
  - 매수하고자 하는 종목의 상승모멘텀이 무엇인지 이해를 하고 매매한다.
  - 나만의 기법을 만들기보다는, 남들이 모두 참고하는 기법을 활용하자!
    - 호가주문창 : 피봇, 저항과지지
    - 이동평균선 : 단기이평은 단순, 장기이평은 지수   cf. 일목균형표에서 9/26/52 참조
