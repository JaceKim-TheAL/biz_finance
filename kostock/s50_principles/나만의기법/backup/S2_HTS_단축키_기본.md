## HTS 단축키(System Short-Key)

### HTS 메인

| 특수키 | 단축키 | 설명  | 비고 |
|:-----:|:-----:|-------|-------|
| | [Home] | 화면검색창 커서이동 | |​
| Ctrl | M | 미니모드 전환(일반→미니) | ⓜ가능화면 |
| Ctrl | O | 일반모드 전환(미니→일반) | ⓜ가능화면 |
| Ctrl | 0 | 신호관리자 팝업   | |
| Ctrl | \ | 화면최소화        | |
| Ctrl | 0 | 신호관리자 팝업    | |
| Ctrl | 1~8 | 가상화면 1~8    | |
| Ctrl + Shift | 마우스 드래그&드롭 | 타이틀바 보이지 않을때 화면창 이동   | 단, 독립실행창에서는 적용안됨 |

### 차트​화면

| 특수키 | 단축키 | 설명  | 비고 |
|:-----:|:-----:|-------|-------|
| Alt   | 1 | 종목메모 호출   | | ​
|   | Space | 연속조회 캔들 수 증가​  | | ​
| Ctrl  | ↑ | 차트 확대    | | ​
| Ctrl  | ↓ | 차트 축소​    | | ​
| Ctrl  | ← | 차트 좌측이동 (캔들 한개씩)​   | | ​
| Ctrl  | → | 차트 우측이동 (캔들 한개씩)   | | ​
|       | ← | 차트 좌측이동 (캔들 20개씩)​   | | ​
|       | → | 차트 우측이동 (캔들 20개씩)   | | ​
| 마우스 | 휠 | 차트 좌우로 이동      | 화면클릭후 마우스 스크롤 | ​
| **[화면편집]** |   | **추세선긋기, 메모, 도형🟨🟢 등 편집** | |
| Shift | A | 수평선​     | | ​
| Shift | D | 추세선​     | | ​
| Shift | S | 추세선 저장 | ⚙️차트환경설정 > ☑추세선 저장 (체크된 경우만) |
| **[차트조회]** |   | **일봉/주봉/월봉/년봉/분봉/초봉/틱 이동** |  |
| Shift | 1,2 | [일/주/월/년/분/초/틱] 좌우로 선택 | | ​
| Shift | 3,4 | [분/초/틱] 상세 주기 이동용       | | ​
