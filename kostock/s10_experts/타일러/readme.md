# 타일러의 단타 타점
> - [동영상](https://www.youtube.com/watch?v=ttmLF7hdHLQ)
<br/>


델타거래량 : [기]체결지수, 매수세 🟧 vs 🟦 매도세

### 테일러의 단타
- 타점만 집중
- 차트는 **횡보장**인지, 횡보가 **돌파**되었는지 딱 2가지로 구분한다
- 차트에서 횡보와 추세비율
  - 구글 제미나이의 답변
  - 추세구간이 약 30%, 횡보구간이 약 70% 차지
- 타일러는 추세구간을 돌파라 칭히고, **횡보구간에만 집중**
  - 돌파가 연이어 발생하면, 그것이 추세를 만드는 것이다
  - 돌파매매는 지양, 왜냐하면 돌파후 휩소가 너무 자주 발생
  - 돌파가 나오면 매매는 소극적, 
  - 돌파 시도후 거절(Reject) 당하는 구간에서 매매는 적극적
- 거래량 지표
  - 긴꼬리 차트에서 거래량만으로는 해석이 모호
  - 저점에서 하단긴꼬리
    - 돌파후 상승세가 강했다는 해석 -> 추세반등
    - 내려오면서 지지라인의 벽을 다 뚫고 내려왔다 해석 -> 추세지속
- 델타거래랑(Delta Volume), [기]체결지수
  - 단순 거래량지표는 디테일이 부족
  - 반면 델타거래량은 가장큰 거래량이 발생한 지점에서 매수세와 매도세 중 어느게 더 강한지 알 수 있다.
  - 15분봉에서 긴꼬리 발생시 추세 예측
