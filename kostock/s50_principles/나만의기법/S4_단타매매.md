### 일봉차트
> 4대일봉자리이며, 이자리 종목만 매매

  ❶ 전고점 도전하러 가는자리 <br/> 
  ❷ 신고가 자리 <br/> 
  ❸ 매물대가 얇은 구간 (빈집털이) <br/> 
  ❹ 바닥 자리 <br/>

### 분봉차트
> 1분봉상 위험한 패턴 TOP5

| 랭킹 | 패턴 | 설명 |
|:----:|-----|-----|
| R1 | ○○빵 패턴      | 뉴스빵 패턴, 최고점일 확률이 높다. 당일 핫한 테마종목만 공략 |
| R2 | 깊은 ○○ 패턴   | 깊은 눌림 패턴, 매물대가 많고 추세전환이 힘듬 |
| R3 | ○가 다 온 패턴 | 끼가 다 온 패턴, 종목마다 다르고 특성이해 필요 |
| R4 | ○파동 패턴     | 3파동 패턴, 마지막 상승은 매수세가 젖먹던힘까지 쏟아냄. 이후 상승여력 약화 (거래량 & 상승률) |
| R5 | ○○○ 패턴      | 가분수 패턴, 윗쪽긴꼬리 Hang in 유형으로 대부분 3파에서 발생, 음봉발생시 바로 탈출 |

### 상승신호탄(상한가분석)
  - 1분봉 : 거래량200% + 상승률1%
  - 3분봉 : 거래량200% + 상승률3%
  - 5분봉 : 거래량200% + 상승률5%
  - **대상승인 경우 거래량이 500%이상**인 경우가 많다.

### 돌파눌림
  - 상승초입 or 돌파타점을 잡지 못하면 **추격매매(뇌동)하지 않는다.**
    - 즉, 계속되는 빨간봉에서 매수금지 (반드시 기준봉을 확인할것)
    - 상승시작과 상승의 끝을 확인후(기준봉), 4등분선 2Q 눌림기다린다.
  - 진짜 돌파는 반드시 돌파선을 지지 확인시켜준다.
    - 반드시 큰거래대금이 동반되어야 한다. 
    - 반드시 매수세의 주체를 확인하자.
  - **상승시작가를 이탈하면 무조건 손절**

### 익절원칙
  - 연속상승시 **거래량이 줄거나, 긴꼬리발생, 음봉**이 만들어지면 무조건 반절 매도(50% 익절)
  - **10일선**이 무너지면 반절 매도(25% 익절)
  - **20일선**이 무너지면 전량 매도(25% 익절)후 매매금지!

### 손절원칙
  - 매수전 손절라인을 긋는다
  - 진행중에 **손절라인을 절대 내리지 않는다!**
  - 매물대를 **돌파하면 손절라인을 올린다.** 
  - 매수직후 **예상과 다르게 움직이면 전량매도** 후 타점을 다시잡는다.
