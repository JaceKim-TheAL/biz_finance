### 마하세븐 어록

- 인공지능/자동매매가 판을치는 시장, 결국 주식을 올리고 패대기치는것은 사람이다. 
- **시장안의 사람의 심리를 읽어야 한다.** 
- 상승1파 눌림목에서 매수, 다음 1파에서 매도
- 0.5%~3% 기계적 매도, 욕심내기보다 기계적매도 
- **투자에서 감정이 들어가면 진다.**
- 원금만 지켜도 돈은 저절로 벌린다.

### 1️⃣ 이평선 눌림목 스캘핑 매매법
- 1분봉 이평선3개 20/50/100 (단지/중기/장기 추세)
- 3개의 이평이 같은방향이면 신뢰도가 높다. 
- 마하이평 수식관리자 
> idx[이평배열_마하지표]

[수식]
```
단기 = MA(가격, 기간1, 이평종류);
중기 = MA(가격, 기간2, 이평종류);
장기 = MA(가격, 기간3, 이평종류);

상승1 = if(단기>단기(1), 1, 0);
상승2 = if(중기>중기(1), 1, 0);
상승3 = if(장기>장기(1), 1, 0);
상승지수 = 상승1 + 상승2 + 상승3;

하락1 = if(단기<단기(1), -1, 0);
하락2 = if(중기<중기(1), -1, 0);
하락3 = if(장기<장기(1), -1, 0);
하락지수 = 하락1 + 하락2 + 하락3;

정배열 = 단기 > 중기 and 중기 > 장기;
역배열 = 단기 < 중기 and 중기 < 장기;
이평배열 = if(정배열, 4, if(역배열, -4, 0));

이평배열
```

[지표조건설정]
```
기간 = [20, 60, 120]
가격 = 종가
이평종류 = 단순
```


- 매수포지션
  - 정배열 : 단기 > 중기 > 장기
  - 주가는 계속 오르는것이 아니라 반드시 조정이 온다
  - **매수타점** : 빠졌다가 **단기이평을 GC** (Golden Cross,상향돌파) 포착하는순간, 매수포지션
  - **손절라인** : 손절선은 **중기이평을 DC** (Dead Cross, 하향돌파)
  - **익절가격** : 익절목표가는 **손절대비 1.5배** 로 설정
  - 단기,중기는 자주발생한다.
  - **장기이평을 이탈한다는 의미는, 장기투자가들까지도 매도세에 합류** 한다는 의미이므로 위험! 
  <br/> ∴ 절대 매수를 한다거나, 물타기는 안됨! 보유하고 있다면 **전량 매도 혹은 비중조정** 은 철칙!

- 스켈핑의 손익비
  - 스캘핑은 한번에 크게 먹는게 아니라, 
  - **짧은 수익을 여러번 누적** 해서 큰수익을 만드는 전략이다!
  - 손익비를 2:1, 3:1 욕심을 내면 승률이 낮아지고, 계좌가 작살난다!

- 매도포지션
  - 매수포지션과 정확히 반대로 생각
  - 역배열 : 단기 < 중기 < 장기
  - 단기이평을 돌파하고 매수하였는데, **단기이평 아래로 이탈하면 손절** 해야 한다. 
  - 휩소로 개미들의 물량을 뺏을수도 있지만, 하락시 크게 얻어맞을수 있다.

### 2️⃣ RSI 눌림목 스캘핑 매매법 
- RSI(상대강도지수, Relative Strength Index)
  - 가격 움직임의 모멘텀(속도와 강도)을 측정하는 기술적 분석 지표
  - 범위 : 0~100 사이의 값, 보통 14일 기준으로 계산
  - 핵심 : 자산이 과매수(70이상) 또는 과매도(30이하) 상태인지 파악하여, 추세반전가능성을 예측하는데 도움
  - 활용 : 
    - 50이상이면 상승추세, 50이하면 하락추세
    - 하락다이버전스, 가격은 상승중인데 RSI가 하락하면 매도신호
    - 상승다이버전스, 가격은 하락중인데 RSI가 상승하면 매수신호

- 기본 개념
  - RSI는 일정 기간 동안의 **평균 상승폭(Average Gain)**과 **평균 하락폭(Average Loss)**을 비교하여 산출
  - 보통 14일 기준으로 계산하지만, 단기 매매에서는 5일이나 7일로 줄여서 사용
- 계산 단계
  - 각 기간의 **가격 변화(Price Change)**를 구합니다.
  - 상승한 날은 Gain, 하락한 날은 Loss로 기록합니다.
  - 일정 기간 동안의 평균 상승폭과 평균 하락폭을 구합니다.

- RSI의 계산식
```
RS(상대강도, Relative Strength) 
RS  = Average Gain / Average Loss
RSI = 100 - (100 / (1 + RS))
    = RS / (1 + RS)
    = AG/AL  /  (AL+AG)/AL
    = AG * AL / AL(AL + AG)
    = AG / (AL + AG)
```
- 의미해석
  - 평균상승폭과 평균하락폭의 합에서 평균상승폭의 포지션을 계산한 지표값으로
  - 평균상승이 높다면 과매수구간으로 인식
  - why? 지속적으로 한쪽방향으로만 상승하거나 하락하는 경우가 작다. 
  - 대부분 파동을 그리며 움직이기 때문이다!


- 단기매매에서 RSI 활용
  - 기본은 14일이지만, 단기는 5일~7일을 자주 사용
  - 단기과열과 침체라인을 80이상/20이하로 좁혀서 사용
  - RSI만으로 페이크가 많으므로, 거래량·이동평균선·캔들 패턴과 함께봐야 한다.
  - 횡보장에서는 신뢰도가 높지만, 강한추세장에서는 과매수/과매도 상태가 오래지속
  - 초단타는 빠른 매수·매도 결정이 핵심이므로, RSI 신호가 나오면 지체 없이 대응하는 습관이 필요

3️⃣ 볼린저밴드 눌림목 스캘핑 매매법

- 상한선과 중심선을 반복적으로 터치하는 것은 상승추세
- 중심선과 하한선을 반복적으로 터치하는 것은 하락추세
- 하락추세에서 추세선을 강하게 돌파할 때가 매수타점
  - 상단 추세 ➔ 눌림목 조정 ➔ 단기 추세선 형성 ➔ 추세선 돌파 

- 자주실수하는 위험구간
  - 상승3파 이후 밴드상한선에 터치하는 경우는 대부분 과매수구간
    - 즉 2번의 상승파동이 나오면 관망
    - 이때 개미들의 매수세가 많고 세력들은 물량을 떠넘긴다. 
  - 상승추세가 꺾이고, 하락추세로 들어가면 매수기회를 포착
    - 하한선과 중심선을 자주터치하며 거래량은 줄어들지만 추세선이 저항대가 되어 터치하면 큰매도물량과 물타기물량들이 계속 나오며 악성저항대가 두터워진다.
    - 하락추세에서 강한 매도세가 나온다면 세력의 이탈일 수 있다. 즉, 당분간 올릴생각이 없단 의미이다.
    - **예측이 아니라, 하락추세선을 강하게 뚫었다가 안착하는 모습을 확인** 하고 들어가야 한다. 
    - 인내가 필요 : 성급하면 먹어도 쬐끔 or 손실, 인내하면 수익 극대화 가능!

### 최종요약

- 스캘핑 매매법 한줄 요약
  - **1분봉 정배열(20>60>120) 초입** ⇒ 단기이평 GC 포착
  - **RSI 과매도구간에 진입** ⇒ 기간은 10, 과매도 45이하에서 상승돌파
  - **하락구간에서 추세선을 강하게 돌파하는 구간** ⇒ 고점을 이은 하락추세선을 돌파

- 매매법의 의미
  - 핵심은 **손절은 짧게 잡고, 익절은 조금 더 길게 (1대 1.5)**
  - 기계적 매수/매도, 망설이는 순간 손실!!!!
  - 감정없이 매매하기 위해서는 **기준이 간단명료** 해야 한다! 지표 최소화!!! 하나의 기법에 맞는 하나의 차트!!
  - 스파게티 차트는 매매를 위한 차트가 아니라, 분석을 위한 차트이다!
  - 반복적인 매매를 통해서 몸이 체득해야 한다!
  - 마하세븐은 고수들의 멘토이다! 이시대 최고의 레전드!
  - 그 누구도 반박하지 못한다. 그 누구도 따라하지 않더라.
