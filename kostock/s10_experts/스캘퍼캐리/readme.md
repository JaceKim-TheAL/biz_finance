### Who is Carry?

- 이력
  - 키움증권 실전투자대회 500리그 우승 (2022년 3차 대회)
  - 소액(500만 원)으로 시작해 3개월 만에 2억 원 달성 사례로 유명
  - 대회에서 1위를 여러 차례 기록하며 “상패도둑”이라는 별명도 있음
- 투자 스타일: 초단타 매매(스캘핑) 중심
  - 호가창 움직임과 거래량 변화를 빠르게 포착해 수 초~수 분 단위로 매매
  - 급등 직전·급락 직후의 순간적인 흐름을 잡아내는 전략 활용
- 매매 철학과 특징 
  | 매매철학 | 특징 |
  |---------|-----|
  | **스캘핑 집중** | 장기 투자보다는 단기 흐름에 집중, 하루에도 수십 번 매매 |
  | **리스크 관리** | 종가 매매(종배)는 피하고, **시초가·급등·급락 순간에만 진입** |
  | **심리적 훈련** | 소액으로 시작할 때는 긴장감이 컸지만, **경험을 통해 흔들리지 않는 멘탈을 강조** |
  | **매매 연습법** | **급등주 호가창** 을 하루 종일 관찰하며 **패턴을 머릿속에 각인** 시키는 방식 추천 |
