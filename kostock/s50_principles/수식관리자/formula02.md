## s[상한D]
> 일봉상 상한가 간 종목의 일자를 빠르게 확인  

[수식]
```js
전일대비등락폭 = C-C(1);
주가변동률 = 100*전일대비등락폭/C(1);

주가변동률 >= 상승률
```

[지표조건설정]
```js
상승률 = 29
```

## s[상한E]
> 일봉상 상한가 터치한 종목의 일자를 빠르게 확인  

[수식]
```js
전일대비등락폭 = H-C(1);
주가변동률 = 100*전일대비등락폭/C(1);

주가변동률 >= 상승률 and H>C;
```

[지표조건설정]
```js
상승률 = 29
```

## s[급등]
> 분봉차트에서 고가기준 상승률 표시
> - 표시:매도, 숫자모양:1~9🟥 표시하면 직관적

[수식]
```js
H상승률 = 100 * (H-O)/O;

범위1 = 급등률;
범위2 = if(변동률>0, 급등률+변동률, 급등률+1);

범위1 <= H상승률 and H상승률 < 범위2

```

[지표조건설정]
```js
급등률 = 1
변동률 = 0
```
<br/>

## s[급락]
> 분봉차트에서 저가기준 하락률 표시
> - 표시:매수, 숫자모양:1~9⬛️ 표시하면 직관적

[수식]
```js
L하락률 = 100 * (L-O)/O;

하락률 = -1*abs(급락률);
범위1 = 하락률;
범위2 = if(변동률>0, 하락률-abs(변동률), 하락률-1);

범위2 < L하락률 and L하락률 <= 범위1

```

[지표조건설정]
```js
급락률 = -1
변동률 = 0
```

## 이평[gc]
> 이동평균선 골드크로스

[수식]
```js
이평 = MA(C,기간);
골든 = CrossUp(C, MA(C,기간));
상돌 = 시가<이평 and 이평<종가;

골든 or 상돌
```

[지표조건설정]
```js
기간 = 10
```

