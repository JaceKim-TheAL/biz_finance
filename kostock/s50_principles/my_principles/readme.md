## 매매기준


### 1. 종목선정
- 매물대 돌파 확인후 진입
- 눌림자리에서 지지를 확인후 진입
- 주요라인에서 터치후 빠지는 종목은 절대 진입금지!!!!
- 추세변곡의 상승초입과 세력의 매집흔적이 확인된 종목

### 2. 매수타점
- 손익비가 좋은자리에서 진입
- 거래량이 급등하면서 상승추세를 만드는 자리
- 트레이들간에 심적으로 유리한자리에 진입
- 조급하고 불안한 자리에서는 잠시도 종목을 보유하지 않는다.

### 3. 손절라인
- 돌파에서는 돌파라인을 이탈하면 손절
- 눌림에서는 직전의 저점을 이탈하면 손절

### 4. 매도타점
- 급상승시 양봉이라도 거래량이 이전보다 줄어들면 매도
- 강한매물대 저항대에서는 미리 매도준비
- 주요저항대는 이평선, 볼벤상한선, 라운드피겨값 

### 5. 종베종목
- 강한테마가 형성된 종목
- 세력의 돈이 들어오고 빠져나가지 않아야 한다.
- 장마감전(오후2시이후) 상승추세이여야 한다.

### 추세매매
> 상승, 하락 아니면 횡보

- 나의 전략은? <br/>
  - 매수한 후 올라갈때까지 버티기 ☹️
  - **안올라가면 약손실 or 본전탈출**  ☺️ 
<br/>

- 매수전 체크 
  | 매수전 체크사항 | 내용설명 |
  |--------|--------|
  | 1️⃣ **지수**     | 종목이 아무리 좋아도, 지수가 하락이면 상승할 확률이 낮다. |
  | 2️⃣ **시황**     | 최근의 뉴스나 어떤 테마가 강한지를 체크, 즉 사람의 관심이 어디에 몰리는지 파악 |
  | 3️⃣ **거래대금**  | 거래량x주식가격, 보통 주식매매시 몇주보다는 총금액을 보고 매매  |
  | 4️⃣ **상승률**    | 거래대금과 상승률을 함께 보면 어떤 종목이 대장주인지 파악이 용이 |
  | 5️⃣ **시가총액**  | 시총에 따라 흐름이 다름, 3천억~10조원 사이 선호   cf.3천억이하는 변동률큼 |
  | 6️⃣ **끼**       | 이종목이 사람의 관심을 받았을때 어느정도까지 올라갈지를 예상 |
  | 7️⃣ **수급**     | 아무리 거래량이 많아도 외국인이나 기관이 팔아버리면 급락 |
  | 8️⃣ **차트위치**  | 일봉부터 확인하여 큰 방향성 파악한다. 신고가돌파 혹은 정배열/역배열 |
<br/>

- 상승추세를 확신하기 위한 조건 
  | 상승추세 조건 | 내용설명 |
  |--------|--------|
  | 1️⃣ **강한수급**        | 매물대, 저항, 전고가 돌파시 강함수급이 동반되어야 확률이 높다.  |
  | 2️⃣ **올라갈자리 체크**  | 윗쪽에 매물대가 2개이상 |
  | 3️⃣ **세력의 매집흔적**  | 주가를 올릴 명분 |
  | 4️⃣ **수급의주체 체크**  | 외국인/기관인지 확인후 당일 수급 확인 |
  | 5️⃣ **확실한 재료**  | 당일 뉴스와 공시, 동종섹터의 흐름 |
  | 6️⃣ **종목의 특성**  | 과거의 급등패턴, 끼있는 종목 |
  | 7️⃣ **시장의 환경**  | 당일 코스피,코스닥 지수 흐름 |
  | 8️⃣ **사람의 심리**  | 시장에 동참하는 사람의 심리 |
