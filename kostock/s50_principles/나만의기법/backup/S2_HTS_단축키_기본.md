## HTS 단축키(System Short-Key)

### HTS 메인

| 특수키 | 단축키 | 설명  | 비고 |
|:-----:|:-----:|-------|-------|
| | HOME | 화면검색창 커서이동 | |​
| Ctrl | M | 미니모드 전환(일반→미니) | ⓜ가능화면 |
| Ctrl | O | 일반모드 전환(미니→일반) | ⓜ가능화면 |
| Ctrl | 0 | 신호관리자 팝업   | |
| Ctrl | \ | 화면최소화        | |
| Ctrl | 0 | 신호관리자 팝업    | |
| Ctrl | 1~8 | 가상화면 1~8    | |

※ Ctrl + Shift + 마우스드래그앤드롭 : 타이틀바 보이지 않을때 화면이동. 단 독립실행창에서는 적용안됨

|:-----:|:-----:|-------|-------|
| Alt   | 1 | 종목메모 호출   | | ​
| Space |   | 연속조회 캔들 수 증가​  | | ​

| Ctrl  | ↓ | 차트 축소​    | | ​

| Ctrl  | → | 차트 우측이동 (캔들 한개씩)   | | ​
|       | ← | 차트 좌측이동 (캔들 20개씩)​   | | ​

[주기이동]
Shift +1,2 (일/주/월/년/분/초/틱) 좌우로 선택
Shift +3,4 (분/초/틱)상세 주기 이동용
​
​
[관심종목]
Insert    : 종목추가​
Delete     : 종목삭제​
Ctrl + Space : 빈칸추가​
Ctrl + `     : 북마크 지정/편집
​
[직장인]
윈도우+D
윈도우+TAB
​
[메로나님 공유]
1. 차트화면에서 space 누르면 조회기간을 늘리는 기능
2. ctrl + 숫자 = 가상화면 전환
3. 관심종목에서 ctrl + ` = 북마크 설정
4. 매수확인창시 space/enter 로 확인 클릭 가능함
5. hts 에서 home 누른 후 숫자로 화면 불러오는 기능 (0339, 4989, 8949, 8282, 0796 등)
6. 4989 키움주문 화면에서 tab 누르면 수량, 가격 입력란 이동
