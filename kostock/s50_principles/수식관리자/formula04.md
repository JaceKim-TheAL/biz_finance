## sw[대장B]
> 끼있는 종목은 과거 대장주 흐름 자주 발생 <br/>
> 단기최저가 이후 급등확률 높음  <br/>
> => 끼있는 종목의 개미털기 단기최저가 반등 공략
> - 대장주 : 1천억이상, 10%이상 상승  OR   상한가
> - 최저가 : 더블비 볼벤(60,2) 하한선 상향돌파 

[수식]
```js
억원 = 100; 
등락률 = 100*(C-C(1))/C(1);
상한가 = 등락률 > 29;

대장주 = amount > 대금1*억원; 
이등주 = amount < 대금1*억원 and 
            amount > 대금2*억원; 

// 더블B하 = BBandsDown(Period, D1);
// 저점 = CrossUp(C, 더블B하);

강세 = (대장주 && 등락률>=상승) or 상한가; 
약세 = (이등주 && 등락률>=상승) and !상한가;

```

[지표조건설정]
```js
- 대금1 = 1000
- 대금2 = 500
- 상승 = 10
- Period = 60
- D1 = 2
```

## sw[수급m]
> - 거래량비율 : 1봉전,2봉전 대비 200% 이상
> - 가격변동률 : 1% 이상

[수식]
```js
거래량비율1 = 100*V/ highest(V, 5, 1);
거래량비율2 = 100*V/ highest(V, 5, 2);
거래량비율 = max(거래량비율1, 거래량비율2);
대량거래 = 거래량비율 > 거래량증가률;

가격등락률 = 100*(C-C(1))/C(1);

강세 = 대량거래 and 가격등락률>가격변동률;
약세 = 대량거래 and 가격등락률<가격변동률*(-1);

```

[지표조건설정]
```js
거래량증가률 = 200
가격변동률 = 1
```

## sw[이평배열]
> 단기/중기/장기 이동평균선의 정배열과 역배열 구간
> - 응용: **쿨라매기 매매기법** 추세매매
>   - 매수 : 정배열(10/20/50)상에서 10선돌파시 매수 
>   - 매도 : 연속상승 후 최초음봉(50%)/10이평(25%)/20이평(25%) 익절

[수식]
```js
단기 = MA(가격, 기간1, 이평종류);
중기 = MA(가격, 기간2, 이평종류);
장기 = MA(가격, 기간3, 이평종류);

정배열 = 단기>중기 and 중기>장기;
역배열 = 단기<중기 and 중기<장기;

```

[지표조건설정]
```js
기간1 = 10
기간2 = 20
기간3 = 60
가격 = 종가
이평종류 = 단순
```

## sw[이평변곡]
> 이동평균선의 본질을 이해하고 변곡점을 감지할 수 있다.
> - 이평선이 모두 상승이면 강세 🟨
> - 이평선이 모두 하락이면 약세 ⬜️

[수식]
```js
상승1 = C >= C(기간1);
상승2 = C >= C(기간2);
상승3 = C >= C(기간3);
하락1 = C <= C(기간1);
하락2 = C <= C(기간2);
하락3 = C <= C(기간3);

상승추세 = 상승1 and 상승2 and 상승3;
하락추세 = 하락1 and 하락2 and 하락3;

```

[지표조건설정]
```js
기간 = [20, 120, 240]
```

## sw[고점저점]
> 신고가와 신저가를 다음봉의 움직임을 분석하여 고점과 저점을 예측한다
> - 신고가를 갱신하면 저항돌파 🟧 
> - 신저가를 갱신하면 지지붕괴 🟦

[수식]
```js
신고가 = 최고값(H, 기간H, 1);
신저가 = 최저값(L, 기간L, 1);

신고돌파 = H > 신고가;
신저붕괴 = L < 신저가;

강세 = 신고돌파;
약세 = 신저붕괴;
```

[지표조건설정]
```js
기간H = 60
기간L = 60
```
