# 상한가매매

### 상한가 의미
  - 상한가는 세력의 개입없이는 불가능하다.
  - 호가창에서 보면 상한가를 지켜주기위해 엄청난 매수수량을 바쳐준다.
    - 누구던 매도해라 다 받아주겠다는 의미
    - 상한가가 안풀리게 계속 받아준다
  - 세력의 의도 
    - 상한가 종가라인은 엄청나게 많은 매수매도세가 존재하는 아주 중요한 매물대이다. 
    - 세력의 물량을 믿고 매수하는 개미들도 많고, 다음날 관심의 대상이기도 하다. 
  - 상한가 다음날 패턴
    - 갭을 많이 띄운후 개미에게 물량을 떠넘기고 빠지는 경우도 많다. 
    - 일부러 이탈시키고 다시 회복후 급등시킨다.

### 상한가 돌파
  - 상한가에서는 엄청나게 많은 매매가 발생하였으므로, 의미있는 라인이다. (저상/지지)
  - 바로 급등할수도 있지만, 개미들을 한번더 털고나서 올라가는 경우도 있다.
  - 상한가 라인을 이탈시키고 다시 올라타는 경우
    - 휩소 즉 개미들의 스탑로스를 이용해 물량을 뺏들고 급등시킨다.
    - 단, 세력의 강한의지가 보여야 확률이 높다. 확인은 거래량!!!
    - 강한지지라인이고 하지만, 이탈시 강한저항라인이기도 하다.
    - 가능하면 상한가이후 20봉 이내에서 공략하기를 권장한다.
  - 상한부근에서 큰거래량함께 다시 상한선을 돌파하면 매수타이밍이다. 

### 수식관리자

- 라인 : ln[상한가라인]
> 수식이름: 상한가L
```
상한가조건 = dayclose()>predayclose()*1.29;
상한가라인 = valuewhen(1, 상한가조건, dayclose());
상한가라인
```

- 신호 : s[상한가GC]
```
상한가조건 = dayclose()>predayclose()*1.29;
상한가라인 = valuewhen(1, 상한가조건, dayclose());
상한가발생 = 상한가라인 > 0;
최근20상 = (BarsSince(상한가조건) >= 2) &&
                (BarsSince(상한가조건) <= 20);
돌파양봉 = (상한가라인 > predayclose()) &&
                (상한가라인 < dayclose()) &&
                (dayclose() > dayopen()) ;
거래량400 = V >= V(1)*4;

신호 =  상한가발생 && 최근20상 && 돌파양봉 && 거래량400;
신호
```
