## 매매포지션

### 1️⃣ 스캘핑
> 매매타임 : 8시10분~10시30분
- **호가창 매물대**돌파 : 큰거래대금과 함께 돌파하는 자리
- **차트분석 매수타점** : **신고가돌파(일봉), 전고돌파(분봉)**
- **시장의 분위기, 사람의 심리** 가 호가에 어떻게 반영되는지를 파악하는게 중요!!!

### 2️⃣ 종가베팅
> 매매타임 : 2시30분~3시30분
- **당일 핫한 테마종목** : 대장주 및 2,3등주가 함께 상승한 테마
- **재료의 연속성** : 새롭게 이슈가 되고 재료의 지속성이 이어질지 판단
- **세력의 매수세** : 외인/기관의 매수 후 빠져나가지 않은 종목

<br/>

### 3️⃣ 시초베팅
> 매매타임 : 8시10분~9시30분
- **신고가 갱신** : 최근 신고가종목이 기간/물량 소화이후 재돌파하는 자리
- **매집의 흔적** : 세력의 매집한 거래대금이 빠져나가지 않은 종목
<br/>

[[TOP]](#index)

### 4️⃣ 단기스윙
> 매매타임 : [월봉차트]월말/월초 or [주봉차트]금요일/월요일
- **추세가 상승전환** : 
  - 이평선 돌파 : [월봉] 10일선 돌파, [주봉] 20일선 돌파
  - 추세선 돌파 : [일봉] 하락추세선 돌파 후 지지를 확인
- **세력의 매수포착** : 개인의 물타기와 손절이 거의 마무리 되고 거래가 거의 없다가 다시 매수세가 들어오는 자리
- **급등후 소화완료**  
  - 최근 관심받은 테마가 급등후 눌림자리에서 장기간 횡보후 횡보박스 돌파
  - 가격조정/기간조정이 충분히 이루어진 자리 (가격대칭/기간대칭)
