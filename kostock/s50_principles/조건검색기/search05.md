### (스윙) [일]3번자리_바닥포착
> 급락후에 장기박스권을 돌파하고 쉬어가는 밥그릇3번자리를 포착
<br/>

- **조건식 생성** : (plus and) 🅿️ / 🅾️ (or group)
| 지표 | 내용 |
|:---:|-----|
| A | [일]1봉전 5봉 평균거래량 50000이상 999999999이하 |
| B | 5일 평균거래대금(단위:백만) 500이상 999999999이하 (금일제외) |
| C | [일]거래량:30000이상 999999999이하 |
| D | 이평이격도[일]0봉전(종가 1, 종가 120) : 97%이상 103%이하 1회이상 |
| E | 주가이평배열(3):[일]0봉전 120이평 < 240이평 < 480이평 |
| F | [일]0봉전 Bollinger Band(20,2) 종가 Bollinger Band 상한선 2%이내 근접 |
| G | 이평이격도:[일]0봉전(종가 1, 종가 5) 2.5%이내 근접 1회이상 |
| H | 가격-이동평균 비교:[일]0봉전 (종가 5)이평 < 종가 |
| I | [일]0봉전 일목균형표(9,26,52) 주가 >= 선행스팬 1 |
| J | [일]0봉전 일목균형표(9,26,52) 주가 >= 선행스팬 2 |


- **조건식 실행** <br/>
__(A or B) and C and D and E and F and (G and H) and (I or J)__

### (스윙) [일]1년신고_눌림20%이상
> _급락후에 장기박스권을 돌파하고 쉬어가는 밥그릇3번자리를 포착_ 
<br/>

- **조건식 생성** : (plus and) 🅿️ / 🅾️ (or group)

| 지표 | 내용 |
|:---:|-----|
| A | 신고가:[일]0봉전 고가가 20봉이내 240봉 신고가 발생 |
| B | 기간내 주가위치:[일]0봉전 10봉 이내에서 종가가 (최고가-최저가) 변동폭이 최저가 대비 10% 이상 |
| C | 52주 최고가대비 -20%이하 |
| D | 신고가:[일]0봉전 종가가 3봉중 신고가 |
| E | 주가등락률:[일]1봉전(중) 종가대비 0봉전 종가등락률 5%이상 |
| F | 거래량비율(n봉):[일]1봉전 거래량 대비 0봉전 거래량 비율 200%이상 |
| G | 신고가:[일]1봉전 종가가 3봉중 신고가 |
| H | 주가등락률:[일]2봉전(중) 종가대비 1봉전 종가등락률 5%이상 |
| I | 거래량비율(n봉):[일]2봉전 거래량 대비 1봉전 거래량 비율 200%이상 |

- **조건식 실행** <br/>
__A and B and C and ((D and E and F) or (G and H and I))__
  - **대전제** : 거래량 증가, 변동률 20%이상, 정배열
  - **이탈여부** : 일봉상 5이평과 장기이평간의 이격도가 5%이내
