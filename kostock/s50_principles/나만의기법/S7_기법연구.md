## 확률높은 시초가 매매

- 종목 발굴 후 구간 분석
  - 월봉 > 주봉 > 일봉 순서로 확률이 높음
  - 주봉에서 갭상승한 종목 중에 중요 라인대(시가, 저항선)를 돌파하는 패턴
  - 중요 라인대를 시가부터 돌파하는 종목들은 눌림목 이후 급등 가능성 있다.
- 매수 타이밍
  - 종목 발굴 후 분봉 상태에서 중요한 라인대를 돌파하는 시점
    - 매수 타이밍으로 잡을 수 있음
  - 그어 놓은 라인대를 손절가로 잡고 뚫리면 칼 손절!
    - 손절가를 짧게 잡아 손절은 최대한 적게, 수익은 크게!
- 재료 분석
  - 종목 발굴 후 재료 파악 꼭 할 것! 
  - 요즘 시장에서 핫한 재료인가? 이슈는?

## 상한가첫눌림 매매
> - 최근상한가 종목중에서 첫눌림구간을 공략하는 단타스윙 (1주이내)
> - 차트에서 수신관리자 만들기 ⇒ 📉[수식관리자](../수식관리자/기법_상한가첫눌림.md)

| 단계 | 매매기법 |
|-----|-----|
| 종목 | 최근상한가 종목 or 장대양봉 종목을 관심종목에 저장 |
| 매수 | 기준봉의 중간값 아래에서 **스토캐스틱슬로우(20,10,20) 과매도구간**을 이탈후 첫상향돌파 |
|     | 30분봉 차트에서 상한가눌림 포착신호 확인 후, 종가에서 매수 |
| 매도 | **기준봉의 (H+L)/2** 에 매도선을 긋는다. <br/> cf. 대략 **17.5% 상승** 지역 |
|     | 1차 매도: 매도선 부근의 라운드피겨값 1~2호가 아래에서 절반 익절 |
|     | 2차 매도: 상승후 추세가 꺾이는 지점에서 전량 매도 (5일선이탈시 or 거래량터진후 첫음봉) |

- 상한가 매매법
  - 매매횟수가 많지는 않지만, 포착되면 2~3일 안에 승부

- 상한가매매 조건4가지 
  - **1. 시장**
    - 미국장 급락 => 국장은 갭하락
    - 갭하락 이후 양봉이 나올 확률이 많음
    - 미국장이 -3%이상 급락한 다음날 기준으로 잡는다.
  - **2. 기업**
    - 상한가 종목중, 저평가 종목은 추가상승확률이 높다
  - **3. 차트**
    - 주봉상에서 전고점 돌파
    - Good !!
  - **4. 재료**
    - 지속적 상승나올수 있는 재료인지 파악
    - 테마주이면 신뢰도가 더 높다.
  - 수익목표 : 10%~15%이상

- Wrap Up

| 기준 | 상한가매매법 |
|------|------------|
| 1.시장 | - **미국 증시가 -3%이상** 급락한 전날 **"상한가 종목"** <br/> - 기업의 아무 악재 없이 시장 때문에 저렴한 구간 형성 ! |
| 2. 기업 | - 상한가를 쳤음에도 **저평가**가 되고 있는 기업! <br/> - 상한가 간 뒤 **시총과 기업의 순현금이 비슷** <br/> - 시장에 Hot한 테마, 종목이 저평가가 되기는 쉽지 않음 <br/> - 상한가 쳤음에도 불구하고 PER, PBR도 저평가 상태 |
| 3. 차트 | - **전고점 첫 돌파 패턴** 등등 <br/> - 추가 상승이 나올 수 있는 차트 패턴을 가지고 있어야 한다. |
| 4. 재료 | - **상승이 추가적으로 나올수 있는 "재료 & 이슈"** |

## 기준봉눌림 매매
> 괜찮은 장목을 발굴해 놓고 눌림목이 나온다면 어떤 시점에 들어가겠다라는 시나리오를 미리 짜져 있는 상태에서 매매

<b>(1) 원칙</b>
- 눌림목 매매
  - 떨어지지 않으면 절대 매매하지 않는다
- 종목발굴법
  - 주봉, 월봉 Good!  첫돌파
  - 재료 파악
- 매매타이밍
  - 눌림목이 나오면 매수
- 매매횟수가 많지 않으면서 더 큰수익 

<b>(2) 실전</b>
- 주봉상에서 
  - 전고점 돌파하는 역망치 캔들
  - 한번도 돌파하지 않는 이평선(10/20/60)을 종가상 돌파하는 역망치 캔들
  - 몸통안에서 매수
- 일봉상에서 
  - 전고점을 돌파하는 자리에서 지지하면 매수
  - 일봉상 중요한 라인을 지지 
  - 주봉의 박스하단에서 갭하락하지 않으면 매수
- 분봉상에서 
  - 쌍바닥을 찍고, 이평선을 돌파후 지지할 때 종가에서 매수
  - 일봉상에서 중요한 라인을 잡았다면, 60분봉상에서 다시 잡고
  - 60분봉상에서 중요한 라인을 잡았다면, 3분/5분 자리에서 다시 매수타점을 잡는다.
  - 분봉상에서 바닥을 찍고 반등 or 그동안 돌파하지 못했던 주요 이평선을 돌파하는 자리에서 매수
  - 몸통을 벗어나면 매매하지 않는다. 
- 손절 및 익절
  - 흔들리지만 주봉의 몸통을 벗어나지 않으면 버틴다
  - 거래량이 터지면서 급등하면 매도 
  - 보통 1~3일이내에서 매도타점이 나옴 (5%~10%)

## 햄버거 매매
> [불개미&대왕개미]님의 유튜버에서 언급한것을 나름 생각하며 정리 <br/>
> • 1분봉 거래대금 차트에서 특정 구간마다 거래량이 뭉치며 위·아래로 층을 이루는 형태 <br/> 
> • 이 모습이 마치 빵(위·아래)과 패티(중간 거래량 뭉침)가 겹겹이 쌓인 햄버거 구조와 유사 <br/>

- 햄버거매매의 특징 

| 구분 | 전략 | 장점 | 위험 |
|-----|-----|------|-----|
| 대장주 매매 | 테마의 중심 종목 매수 | 안정적, 갭상승 가능성 큼 | 진입 타이밍 놓치면 기회 제한 |
| 후발주 매매 | 대장주 급등시 따라붙는 종목 매수 | 단기 급등 가능성 | 대장주 하락시 급락 위험 | 
<br/>

- 햄버거매매의 적용 및 유의사항 
  - 데이트레이딩에서 적용 (일봉X, 스윙X)
  - 대장주 위주 매매
    - 테마주가 움직일 때는 웬만하면 대장주를 공략
    > → 대장주는 상한가에 도달하면 다음날 갭 상승을 줄 가능성이 높다.
  - 후발주(이등주·삼등주) 매매 조건
    - 대장주가 너무 빨리 상한가에 도달해 기회를 주지 않을 때만 후발주를 공략
    > → 하지만 대장주가 조금만 흔들려도 후발주는 크게 무너질 수 있어 위험
  - 갭 상승 종목 주의
    - 전날 시간외 거래에서 크게 오른 뒤, 다음날 또 갭을 띄운 종목은 매매하지 않는 것이 원칙
    > → 변동성이 크고 위험하기 때문
<br/>

- 종목 선정 기준

| 선정기준 | 상세내용 |
|---------|--------|
| 테마 강도 | 뉴스, 정책, 이슈로 인해 강하게 움직이는 테마가 있는지 확인 <br/> → 예: 로봇, AI, 원전, 정치 관련주 |
| 대장주 확인 | 테마 내에서 가장 먼저 움직이고 거래량이 몰리는 종목 <br/> → 대장주가 햄버거 패턴을 만들면 신호 강함 |
| 거래량 집중 | 1분봉 기준으로 거래량이 중앙에 뭉쳐 있는지 확인 <br/> → 위·아래 거래량이 얇고, 가운데가 두꺼우면 햄버거 패턴 |
| 호가창 흐름 | 매수세가 강하게 들어오는지, 호가창이 밀리지 않는지 체크 <br/> → 실전에서는 호가창이 가장 중요한 판단 기준 중 하나 |

- 햄버거매매 일지작성
> 햄버거매매는 단타중심이기 때문에 빠른복기와 반복학습이 중요

❶ 기본 항목 기록 

| 항목 | 내용 |
|-----|-----|
| 날짜 | 매매한 날짜 |
| 종목명 | 매매한 종목 |
| 진입가/청산가 | 매수·매도 가격 |
| 수익률 | 실현 손익과 수익률 |
| 거래량 패턴 | 햄버거 패턴 여부 (O/X) |
| 대장주 여부 | 해당 종목이 테마의 대장주인지 |
| 매매 이유 | 진입 근거 (패턴, 뉴스, 수급 등) |
| 복기 | 잘한 점 / 개선할 점 |

❷ 패턴 캡처

  - 1분봉 차트에서 햄버거 패턴이 보이는 구간을 이미지로 저장해두면 복기에 큰 도움이 된다.

❸ 실수 기록

  - 손실이 났을 경우, 왜 손실이 났는지를 구체적으로 기록
  - 예: “대장주가 흔들렸는데 후발주에 물림”

### ABCD 매매
> 차트의 흐름을 특정구간으로 나누어 매매타이밍을 잡는 기법

- ABCD 매매의 기본 구간
  - A 구간: 초기 상승 시작점
  - B 구간: 상승 후 첫 번째 고점 형성
  - C 구간: 되돌림(조정) 구간, 눌림목 매수 기회로 보는 경우가 많음
  - D 구간: 다시 상승하는 구간, 목표가 설정 및 매도 포인트

- ABCD 매매는 **상승 → 조정 → 재상승의 패턴**을 차트에서 반복적으로 찾아내는 방식이다.
  - 일부 자료에서는 A~F까지 더 세분화하기도 하지만, 기본적으로는 ABCD 네 구간으로 설명된다.
  - 흥미로운 점은 이 방식이 단순히 "패턴을 외워서 따라간다"는 게 아니라, **시장 흐름을 단계별로 인식하고 심리적 전환점을 매매 포인트로 삼는 것** 이라는 점이에요.

- 단타에서 ABCD 매매 활용 포인트
  - **A → B 구간: 초기 급등 구간**. 단타에서는 이 구간을 놓치면 뒤늦게 추격 매수하기보다 눌림목(C)을 기다리는 게 안전
  - **C 구간: 조정·눌림목.** 단타 매수 포인트로 가장 많이 활용되는 자리. 거래량이 줄면서 가격이 안정되는 순간을 노려 진입.
  - **D 구간: 재상승.** 단타에서는 이 구간에서 빠르게 익절하는 경우가 많다. <br/> 목표가를 짧게 잡고, 재차 고점을 돌파하는지 여부를 확인

- 단타 매매 시 주의할 점
  - **거래량을 확인**: 단순히 패턴만 보는 게 아니라, **거래량이 줄었다가 다시 붙는 순간이 핵심** 신호
  - **손절라인 설정**: 눌림목(C)에서 진입할 때, **직전 저점을 깨면 바로 손절**하는 규칙을 두는 게 중요
  - **빠른 대응**: 단타는 몇 분~몇 시간 단위로 움직이므로, **ABCD 패턴이 무너질 경우 즉시 대응**해야 합니다.
  - 즉, 단타에서는 **C 구간 매수 → D 구간 익절**이 가장 많이 쓰이는 전략

- 일봉/분봉 병행 활용 전략

  - C 구간 눌림목에서 거래량과 캔들 패턴을 세밀하게 확인

| 일봉 차트 | 분봉 차트 (1분, 5분) |
|----------|--------------------|
| √ 전체 추세와 **주요 지지·저항 구간** 확인  | √ 실제 **진입·청산 타이밍**을 잡는 데 활용 |
| √ ABCD 패턴이 형성되는 **큰 흐름**을 파악  | √ **C 구간 눌림목**에서 거래량과 캔들 패턴을 세밀하게 확인 |
| √ **매매 방향**(상승 추세인지, 단순 반등인지)을 결정 | √ **D 구간에서 익절 타이밍**을 빠르게 포착 |


- 단타 ABCD 매매 실전 팁
  - 일봉에서 ABCD 패턴 확인 → 분봉에서 진입 타이밍 잡기
  - 예: 일봉에서 A-B-C-D 구조가 보이면, 분봉에서 C 구간 눌림목을 캔들·거래량으로 확인 후 매수.
- **거래량 급증 신호** 활용
  - 분봉에서 거래량이 갑자기 붙는 순간은 D 구간 진입 신호일 가능성이 큽니다.
- **익절은 짧게, 손절은 확실히**
  - 단타에서는 목표가를 짧게 잡고, 손절 라인을 미리 정해두는 게 필수
  - 즉, 일봉으로 큰 그림을 보고, 분봉으로 칼같이 대응한다는 게 핵심

- 단타에서 자주 쓰이는 캔들 패턴 + ABCD 매매

### HTS차트설정
> 단박지표, 수박, 레인보우, 무지개, 파란점선 세력문 화살표 볼린저밴드 일목균형지표 등

- 기본적인 이동평균선의 차트에서
  - [1]지분기법, [2]대칭이론, [3]기간대칭을 본다
    - 224일선이나, 112일선을 기준으로 매수의 힘이 쎈지, 매도의 힘이 쎈지
    - 앞전에 나온 파동으로 이후의 파동이 어디까지 이어질지, 어디까지 빠질지, 어디까지 오를지 예측
    - 오르는데 3만큼의 길이가 나왔다면, 다음 눌림목으로 어디까지, 언제까지내려갈지 예측
  - 5일선 방향선, 20일선 추세선, 60일선 수급선, 120일선&240일선 세력선
  - 일목균형표에서 구름대 아래에 있으면 힘이없고, 구름대 위에 있어야 힘이 있다.
  - 볼린저밴드는 주가가 다니는 길
    - 위로 뚫는 우수생이 나오면, 곧 볼린저 상단이 뚫리니 응축된 힘이 터지는 과정에서 주가가 상승하며
    - 상승이후의 추가 상승전 눌림목을 잡을 수 있는 하단선이 지지라인
  - 파라볼릭지표에서 금밭구간에선 주가가 힘이 있고, 은밭구간에선 주가가 힘이 없다.
  - 엔벨로프에서 변동성이 큰종목이 지지선을 터치하고 반등하는 낙폭과대 종목을 포착
