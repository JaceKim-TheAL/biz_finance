### 돌파스켈핑
- 검색포인트
  - 3분봉 130 이동평균선은 하루 평균 추세를 반영하며, 단기 상승 흐름을 포착하는 데 유용

- 조건식생성

| 조건항목 | 상세설명 | 
|--------|--------| 
| 주가 이동평균 추세  | 3분봉 기준, 130 이동평균선 상향 돌파  | 
| 체결강도           | 100% 이상 (매수세 우위)             | 
| 호가잔량비         | 매도잔량 ≥ 매수잔량의 200% 이상       | 
| 거래대금           | 일 기준 100억 이상                  | 
| 시가총액           | 800억 ~ 5000억 사이                | 
| 당일 고가 돌파 여부 | 현재가가 당일 고가를 돌파한 시점       |

### 종베스켈핑
- 검색포인트
  - 이 조건은 당일 강한 상승 흐름을 보이며 종가에 매수할 만한 종목을 선별하는 데 적합


- 조건식생성

| 조건항목 | 상세설명 | 
|--------|--------| 
| 당일등락률          | +3% 이상 (양봉 기준)               |
| 거래량 증가율       | 전일 대비 150% 이상                |
| 거래대금           | 일 기준 50억 이상                  | 
| 체결강도           | 110% 이상 (매수세 우위)             | 
| 당일 고가 근접 여부 | 현재가가 당일 고가 대비 95% 이상      |
| 이동평균선 위치     | 5일선, 20일선 위에 위치              |
| 시가총액           | 800억 ~ 5000억 사이                |
