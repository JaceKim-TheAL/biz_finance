## 나만의 기법

### 나의전략
- **위험관리매매** : **수익보다 손실최소화**, 매수즉시하락 or 3분이내 상승하지않으면 바로 매도(재타점), **안올라가면 약손실 or 본전탈출** 
- **추세반전매매** : **추세반전(하락추세선돌파,공구리돌파) 초입**, 전고점부근 분할매도, 터치하고 돌파못하면 전량매도
- **주도대장매매** : **포착종목(검색기/관종) 종목섹터의 대장주 공략**
- **돌파혹은눌림** : 시간이 오래걸리는 **~횡보구간(박스권)에서 매매금지~**, 즉 **방향성을 확인하고 짧고 빠른 승부!**

### 매매전략 
- **돌파매매** : (확인) 전고점 및 신고가 직전에 **강한거래대금과 수급이 포착**되는 종목
- **돌파눌림** : (예측) 돌파이후 돌파자리 혹은 돌파봉의 시가에서 **지지확인 후 눌림자리**를 공략 (엘리어트 상승1파/3파)
- 시간은 기회비용, 길게 크게 먹기보다 확실한 자리에서 빠르게 승부!!

### 종목선정 
- **시장 주도주** : 테마종목중 **대장주(2,3등주와 함께 테마군이 상승인 대장주)**
- **끼있는 종목** : 최근6개월 이내 **1천억이상 장대양봉(상승률>20%)** 2번 이상
- **검색기 포착** : 바로 매수하지 않고,아래 프로세스대로 **기계적으로 빠르게 체크**
  - [x] ❶ 포착종목의 섹터 및 대장주 확인 : 거래대금상위 종목 및 섹터그룹별 종목을 빠르게 확인  
  - ❷ 재료 확인 : 공시 확인 및 뉴스 검색
  - ❸ 상승공간(저항선거리) 확인 : 차트(주봉/일봉)의 전고점, 신고가 근접
  - ❹ 손절가 확인 : 의미있는 매물대, 지지(전저점/돌파전저항선)선 이탈시 손절 
  - ❺ 대비 1.5배 이상일때만 매수포지션
  - ❻ 거래원분석 : 마지막으로 매수주체가 이탈하지 않은경우에 매수 (프로그램매매/거래원분석) 
  - **테재공손대거** : 테마/재료/공간/손절/대비/거래원

### 종가베팅(단스)
- 당일 관심종목중에서 **2시이후 상승여력이 남아있는 종목**
- **재료가 살아있고,** 지속적으로 주가를 끌고갈수 있다고 판단되는 종목
- 단, **상승장일때만 종가 진입** or **선강후약 종목** 에만 진입
- 관심종목 정리 : ❶상한가 ❷대금탑 ❸신고가 ❹신고거 ❺돌파봉 ❻눌림목 ❼뉴스시황

|:----:|---------|---------|

| 🔴 | **정부 정책 수혜** | **특정 산업에 대한 지원, 규제 완화, 세제 혜택** 등이 해당 업종 주가를 밀어 올림 |

| 🟡 | **기술 개발/혁신** | **신기술 발표, 특허 획득, 임상시험 성공** 같은 소식은 투자자 기대감 상승 |

  - 반대로 3가지가 동시에 맞물리면 강한 상승세, 확신을 가지고 매수
